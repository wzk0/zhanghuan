# 掌环 (zhic_tool)

一个方便天津理工大学中环信息学院的同学们使用的app, 支持**查课表**, **查空教室**, **查成绩**, **查考试信息**; **支持选课**, **评教**, **智慧宿舍请假**. 遵循 `Material Design 3` 设计规范, 使用 `flutter` 进行开发.

## 安装

从[最新的Release](https://github.com/wzk0/zhanghuan/releases/latest)中下载 `arm64-v8a apk` 安装包即可.

> 其他架构自行选择.

## 开发与构建

若要对`掌环`进行二次开发, PR或自行构建:

1. fork `https://github.com/wzk0/zhanghuan.git`;

2. clone;

3. 确保你的PC上有 `flutter sdk`, `vscode` 或 `android studio`;

4. 运行 `flutter pub get`;

5. 选择调试设备为安卓后, `flutter run`.

> 目前掌环仅在安卓设备上测试过, `Linux` 可能有些问题, `Web` 不可用.
